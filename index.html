Get Set Go

Get Set Go

Get Set Go is a visual-musical VR experience.


Get Set Go is built with Unreal Engine 4 and is based upon an electronic music production of mine. Events are synchronized with the various sections of the song through use of Sequencer, blueprint interfaces, collision volumes etc.

Visual effects were developed through leveraging material parameters, various interpolations, event driven functions, and more. An Audio Visualizer plugin enables the ability to dynamically calculate audio frequencies based on current track playback, which can then be fed into various visual feedback.

For optimization purposes, there is utilization of level streaming, collision management, tick management, and more. Performance is always a priority for Virtual Reality in order to facilitate a smooth experience for the player.
